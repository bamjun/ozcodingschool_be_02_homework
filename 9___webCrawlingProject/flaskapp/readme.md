- 플라스크 앱 세팅하기  
  ```cmd  
    set FLASK_APP=index  
    ```  
<br>  

- 플라스크 실행해보기  
  ```cmd  
    flask run  
    ```  

<br>  

- 플라스크 디버그 모드로 세팅  
  ```cmd  
  set FLASK_DEBUG=true  
  ```

<br>

- 플라스크에서 db 사용하기 위해 ```flask-migrate``` , ```Flask-SQLAlchemy``` , ```pymysql``` 설치  
  ```cmd  
  pip install flask-migrate Flask-SQLAlchemy pymysql  
  ```  

<br>  

- db초기화  
  ```cmd  
  flask db init  
  ```  

<br>

- 리비전 파일 생성하기  
  ```cmd  
  flask db migrate  
  flask db upgrade  
  ```
